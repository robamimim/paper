# 発達障害の私の時間

私は子供の頃から遅刻してばかりだった。クラスに遅刻の回数を張り出され自分だけが天井に届いていた。不良品と呼ばれ学校からはじかれた。働きはじめてからも、遅刻、遅れ、早すぎと、時間には苦労しつづけた。

ただ幸いにも、世の中には様々な場があり、うまくやっていける場もあった。

しかし時間の混乱はさまざまなトラブルの元になる。電車に遅れ、飛行機は飛んでいく。〆切に遅れ、会議に遅れる。お鍋はどれも真っ黒焦げ、デートに遅れる。「時は金なり」そんなビジネス書が山のように積まれているのを見て逃げ出した事がある。

そのうちADHDのブームがやってきて、診断を受けたら正真正銘のADHDとのことだった。これは努力や根性では治らないので、現実を受け入れてうまく生きて行きましょうということになった。

そこから試行錯誤の日々がはじまった。

今回の卒論では成人のADHDの時間をめぐる問題とテクノロジーで解決する方法をテーマにする。まずは一つの事例として、私の試行錯誤を振り返ってみたい。

# 手帳、Googleフォーム、Googleスプレッドシート

時間の記録をつけ始めたのは、2000年代だった。ITバブルの中、どこも人手不足で大目に見てもらえる環境にもぐりこんだ。トラブルは無茶な働きでカバーして成果を上げていた。

そんな時、私が時間に苦労しているのを見て「1分単位で記録をつけよう」とアドバイスをくれた人がいた。自分もそうであったからと励ましてくれた。方眼紙のノートを買って24時間のマークをつけ、そこに何をしたかを書いていった。

まず見えたのが、さぼってしまう時間が思った以上に長いという事だった。自分が嫌になって仕方がない。だけど「良いか悪いかは置いておいて記録しよう」と励まされた。誰にも絶対に見られないようにして、できるだけありのままを書くようにしていった。

正味の活動時間は１日のうち2、3時間だった。さぼることが3時間、書類や準備の時間が３時間。お昼１時間夕食１時間。12時間職場にいて中身はこうだ。

特に嫌なのが書類の作業だった。手に持つだけで嫌な気持ちになり、フッと時間が飛んでしまう。自分がダメなやつに思えて仕方がなかった。ところがこっそり周りの人の時間を測ってみると、上司の皆さんも案外そんなものなのだった。しかしうまくやっている。

遅刻をしない、時間を守ると百難隠すのだ。ならば決して遅刻をしないようにしよう。固く決意した。遅刻が減った。十中八九はなんとかなった。しかし十中二三は遅れてしまう。

こんなにどうにもならないのに、毎日、同じパターンを繰り返している。同じ時間に集中し、同じ時間に休憩をとり、同じ時間にお腹がすき、同じ時間に眠くなっている。不思議だ。いったい何故こうなるのだろう。

また、遅刻する時は疲れている時、失敗が連続する時は、必要な時間が取れていない事に気がついた。嫌で嫌で仕方がない仕事が正味１分で完了しているのを見つけた時は、思わず笑ってしまった。

そのうち手帳を取り出して書くことが面倒になり、一旦このことを忘れてしまう。

数年後また時間に追われるようになった。今度はスマホ用にGoogleフォームでスプレッドシートに入力することにした。そして後から見返す。相変わらずなんでこんな事ができないのかと自分が嫌になる。しかしスプレッドシートは集計ができるのだ。そうすると、がんばっている自分が見えて元気になれた。

集計する事で気がついた。なぜか自分に無理な要求をしてしまっている。24時間あっても出来ない量の仕事を引き受けていた。

そして疲れると、どんどんトラブルが増え、手に負えなくなって破綻する。この先も自分の人生はそういうことの繰り返しなのだろうか。

再び、だんだん無理が重なり退職した。また記録をしなくなった。

卒論に取り組むまですっかり忘れていたことだ。

# タイマーとストップウォッチ

そして数年前、またしても仕事でトラブルが積もり限界を感じて退職し、しばらく休む事にした。もうこんな事は繰り返したくない。時間の記録を再開した。

なぜ生きてるだけで時間がかかるんだろう。本やインターネットでお手本を探すと、皆さん朝は起きて１時間で家を出ているのだ。私はどうやったって２時間かかる。なぜだろう。身だしなみを整え、部屋を綺麗にして、朝食をたべる。そうすると朝からもう疲れている。だったらギリギリまで寝てダッシュで出かけた方がまだマシだ。

ストップウォッチを持って測りまくった。朝起きて歯を磨いて顔をあらう。洗濯して掃除して朝ごはんを食べる。服を着て今日の出かける支度をする。

その中で、たびたび夢中になって時間が飛んでいる事に気がついた。次の予定の事などすっかり忘れてしまっている。反対に１０秒で満足することも、すっかり忘れている事もあった。

そこで、タスクリストを作り、3分にタイマーを固定して、歯を磨く、髪を整える、服を着ると時間を区切り、その間はそのために時間を使うと決める事にした。この３分は歯のため。確保。他のものは寄せ付けない。鉄壁の守りのイメージを作りそれを積み重ねた。

並行して、部屋の模型を作って最短の動線を考えてリフォームした。朝のタスクリストはパネルにして目立つ所においた。

それでも１時間半かかって朝からぐったり疲れてしまう。これは私自身がお手本のように器用ではない、ということなんだろう。シェアハウスに住んでいた頃を思い出した。ものすごく手際がよい人もいれば、ゆっくりの人もいた。すごい人、あの人のようには出来ない。

そこで、自分にできるペースで、10分、30分、60分のコースを考えた。

起きたら時間を確認して、出かけるまでの時間を引き算してタイマーを起動し、タスクリストを見ながら支度する。これはいい感じだった。だんだん暮らしが快適になってきた。また、毎日意識して同じ事を繰り返すと手際もよくなるようだ。

タイマーを使って生活しているうちに、時間が「見えて」くるような感覚があった。

タイマーがないと手探りで間違いながら進む感じだ。暮らしが快適になってきた。遅刻がかなり減った。休む時間、必要な時間を取れるようになった。

そして普通の人たちはタイマーを使わなくても時間がわかるのか、と驚いた。

しばしばタイマーをかけ忘れると全く元の通りで時間がすっ飛んでいるのだ。

「時間の見当識」が欠けているのかもしれない。タイマーは時間の見当を補うことができるのだ。

# IoT照明

朝の時間は対策ができた。次に困っていたのが昼間の活動だ。面白くなるとギリギリまで、時には破綻するまでやめられない。タイマーが鳴っても気が付かないのだ。

これは大好きなマンガからヒントをもらった。
￼
本好きの下剋上4-12

時間になると照明が徐々に暗くなるようにした。これは画期的だった。どんなに集中していても光がなくなると、作業を終えられるのだ。

朝もアラームから光に変えた。明るくなると自然に目が覚める。

作業する時間は白い光、リラックスする時はオレンジの光と設定した。こうすると気持ちの切り替えが楽にできる。

時間になると作業を始め、時間になると作業を途中で辞める。途中で辞める事に慣れてきた。今までは完全に満足するまで辞められなかったのだ。

# Evernote、Googleカレンダー、壁掛けカレンダー

だんだんと１日がどんな姿をしているか見えてきた。最初はEvernoteにいろいろなメモをして、それに日付をかいていった。そのうち未来の日付を書いて、アラームが鳴るようにした。さらにGoogleカレンダーと連携して予定を一覧できるようにした。しかしアラームが鳴るまで予定の事をすっかり忘れてダブルブッキングしてしまう。

そこで壁掛けのカレンダーにも予定を書くことにした。年間カレンダーなので当分先まで見渡せる。これを毎日確認する。だいぶミスが減った。

これでもミスがまだ起きる。仕方がないので大事な予定は誰かにお願いして知らせてもらう事にした。放送大学の障害学生支援係には大変助けていただいている。

# Todo list
Evernoteを愛用してきたが、サービスが変わって自分に合わなくなってきた。そこでいくつかのアプリを試して、今はTodo List というシンプルなタスク管理ツールを使っている。

何かやろうと思ったら必ずTodo Listに記録してから着手する。最初は面倒だが、これのいいところは繰り返しのタスクを自動化できる事だ。

毎日、毎週、毎月、毎年のタスクが洗い出されてくると、生きてるだけで、ものすごい量のタスクがあることに圧倒された。

恐ろしい事に今まではトラブルになってからやっていたのだ。電気やスマホが止められる事もあった。トラブルというのは優先順位が最高度で可視化される現象なのかもしれない。

毎日、毎週、毎月、毎年のタスク。
大事な事、急ぎの事。

特に大事なのが繰り返すタスクだ。あらかじめ使える時間がわかる。1日24時間からタスクの時間を引いたものが活動可能な時間だ。その中でできる事を考える。24時間あってもできない事は、引き受けない。

するとかなり先の事まで考える事ができるようになってきた。

これとカレンダーとあわせて長い時間の見当識を支えるツールになった。

そして予定が立てられるようになり、トラブルが減りつつある。

# Toggle track
あるときカウンセリングにて生活記録表をつける事をすすめられた。睡眠、食事、その日の調子、活動などをタイムテーブルに書き込んでいくものだ。

ずっと以前に方眼紙につけていた時間の記録と同じだ。最初は紙の記録用紙につけていたが、夜になるとその日一日何をしていたか忘れている。

ある時、スマートフォンのアプリのカテゴリに「ライフログ」があるのを見つけた。生活記録表だ。

これだとスマホから入力できるので、行動したその場で入力できるのがいい。いくつか試してToggle trackが気に入った。睡眠、体調、仕事と勉強、家事や書類、健康と運動、趣味や遊びの6色にわけている。

そうすると、今日はなんにもやってないと焦る気持ちの日もそれなりに動いていたり、さぼってばかりの日も、その日は休む事が必要だったと分かるようになってきた。　

自分の中の時間と実際の時間にずれがあるのだ。今はこれを元に認知行動療法のカウンセリングを受けている。

最近はライフログに今日１日の予定を入れる事ている。そうすると予定の変更や入れ替えが楽にできるので、調子の悪い日でも、悪い日なりにベストな１日を過ごせるようになりつつある。

# まとめ
タイマーで時間を確保し、ストップウオッチで時間を測り、IoT照明で区切りをつける。ライフログに記録をつける。繰り返すタスクをTodoリストに、予定をGoogleカレンダーと壁掛けカレンダーに。大事な予定は人に頼む。

この取り組みを続けるうちに、時間が見えるような、１日の形が見えるような、感覚がしてきている。

明日の事を想像できるようになってきた。今までは明日とは唐突に訪れるものだった。１週間が見えると来週の自分を想像できる。実のところ一ヶ月先はまだぼんやりしている。一年先は真っ暗闇だ。

これら支援とツールのおかげで、だんだんスムーズに暮らせるようになってきた。

しかし、いまだに予定を決めて行動することが苦手だ。リラックスして楽しんでいると次の事を忘れてしまう。

旅行が好きだが、いつも当日券を買っている。予定を決めない旅行は自由でいいねといわれるが、それしかできないのだ。

時間の見当識とは私にとってSFのようなものだ。

テクノロジーの支援なしに暮らしている人は、一体どうなっているのだろう？ワーキングメモリーが生活の維持管理に割り振られているという事なのだろうか？普通の人の暮らしが謎である。

# 結論
こうして振り返ってみると、時間感覚に乏しいため、時間ベースで作業する経験が不足し、時間の取り扱いが下手なまま、トラブルを繰り返すという負のスパイラルに陥っていたのではと考えられる。

この経験を素材にして卒論に取り組みたいと考えている。
